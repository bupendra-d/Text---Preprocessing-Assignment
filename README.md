# Text Analytics Assignment

This assignment is based on basic **NLP and Text Analytics** concepts in Python.

## What is done in this project?
- Performed text pre-processing
- Applied **TF-IDF** for feature engineering
- Calculated cosine similarity between 3 sentences
- Implemented **Logistic Regression** for text classification
- Implemented **K-Means** for text clustering

## Tools & Libraries
Python, Pandas, NLTK, Scikit-learn
